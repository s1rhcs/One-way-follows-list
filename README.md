# Fake Ass Fucks List

A simple, private tool that shows who you follow on Instagram that **don’t** follow you back. Everything runs in your browser on your own computer. No login, no uploading data to a server. [web:258][web:261]

---

## What you need

- An Instagram account.
- A computer (Mac or Windows) with:
  - A web browser (Chrome, Edge, Firefox, or Safari).
  - The Instagram data **ZIP file** downloaded from Instagram. [web:258]

---

## Step 1 – Download your Instagram data

1. Open a browser and go to **instagram.com** and log in. [web:258]
2. Click your profile picture → **Settings**.
3. Go to **Accounts Center** → **Your information and permissions** → **Download your information**. [web:258]
4. Choose:
   - **All available information** (or at least “Followers and following”).
   - **Format: JSON**.
   - Request the download.
5. Instagram will email you a link. Click the link and download the **ZIP file** to your computer (for example, to your Downloads folder). [web:258]

---

## Step 2 – Find the followers and following files

1. Locate the ZIP file you just downloaded (usually in **Downloads**).
2. Double‑click the ZIP to extract/unzip it.
3. Open the extracted folder, then look for a path similar to:

   - `connections` → `followers_and_following` →  
     - `followers_1.json`  
     - `following.json` (or `following_1.json` for larger accounts)

4. Remember where this folder is. You will need **followers_1.json** and **following.json** for the tool. [web:261]

---

## Step 3 – Open the tool

### If you are using a website link

- If this tool is hosted (for example on GitHub Pages or Netlify), just open the link in your browser and skip to **Step 4**. [web:247][web:248]

### If you downloaded the HTML file

1. Save the file as `index.html` (or `new_following.html`) on your computer.
2. Double‑click the file:
   - On **Mac**: it should open in Safari or your default browser.
   - On **Windows**: it should open in Edge, Chrome, or your default browser.
3. You should see a page titled “Fake ass fucks” with an upload area and a **Build fake ass list** button.
